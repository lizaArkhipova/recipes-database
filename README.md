# Culinary Recipes Database (C)

## Описание проекта
Консольное приложение на языке C для хранения и управления базой кулинарных рецептов.  
Проект реализует добавление, удаление, поиск, сортировку и сохранение данных в файл.

## Функциональность
- добавление рецептов
- логическое удаление записей
- поиск по названию и ингредиентам
- сортировка по алфавиту
- сохранение и загрузка данных

## Используемые технологии
- язык C
- структуры данных
- работа с файлами
- алгоритмы сортировки
- Linux

## Запуск
```bash
gcc recipes.c -o recipes
./recipes
```

## Тестирование
В процессе разработки проводилось ручное тестирование:
- проверка добавления пустых и некорректных данных
- поиск по несуществующим ингредиентам
- удаление по невалидному ID
- проверка сохранения и загрузки данных
- проверка сортировки при наличии удалённых записей
